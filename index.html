Claude Opus 4.5 : Anthropic reprend la couronne du code

Anthropic lance Claude Opus 4.5, présenté comme le meilleur modèle au monde pour le code et les agents. Il atteint 80,9% sur SWE-bench Verified, surpassant GPT-5.1 et Gemini 3. Les prix ont été divisés par trois.

Runway Gen-4.5 domine les benchmarks vidéo

Gen-4.5 atteint le score le plus élevé jamais enregistré (1,247) sur le benchmark Artificial Analysis Text-to-Video, devant Veo 3 et Sora. Le modèle excelle dans la précision physique et la cohérence des matériaux.
